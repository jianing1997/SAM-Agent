# Data and Model Availability

This repository contains the public source code for SAM-Agent. Large files are intentionally excluded from GitHub to avoid Git LFS bandwidth limits and to keep the code repository lightweight.

## Included in This Repository

- SAM-Agent source code
- Model training, evaluation, and analysis scripts
- Web application code
- Environment specification
- Example workflow notebook
- Small configuration templates

## Excluded from GitHub

- Full generated SAM molecular libraries
- Training and evaluation datasets
- Trained model checkpoints and weights
- Retrosynthesis datasets and one-step model resources
- RAG vector indexes and generated runtime artifacts
- Large video, PDF, and figure-generation outputs

## External Archive
